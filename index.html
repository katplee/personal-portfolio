On a quest to become the greatest warrior ever, the main hero tames and fuses with dragons to become stronger. A simple RPG with turn-based fighting with enemies in-game!
Tasks
Joined a team from South America and fought through a 14-hour time difference to create a playable RPG game for the Brackey's Game Jam 2021.1 where I wrote almost 95% of the code. Built the systems for all game elements, implemented player movement, created the inventory system with scriptable objects, scripted the whole turn-based combat system, designed the save and load system between scenes using serialization and implemented the update system for the HUD and panels for all scenes.
